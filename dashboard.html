    About This Dashboard

    This dashboard presents relative vegetation condition at ward level for Kakamega County, Kenya. It is designed to support planning and situational awareness by county governments and climate programmes.

    What the colours mean

    Normal — vegetation is at or above typical levels for this ward and calendar month.
    Watch — vegetation is moderately below historical norms.
    High Stress — vegetation is significantly below historical norms.

    Data source

    Sentinel-2 L2A satellite imagery (10m resolution, Level-2A surface reflectance) accessed via Microsoft Planetary Computer. NDVI (Normalized Difference Vegetation Index) is computed from the Near-Infrared and Red bands, aggregated as monthly median composites with cloud masking applied via the Scene Classification Layer.

    Ward boundaries are based on Kenya's 60 administrative wards within Kakamega County.

    Classification method

    Each ward's NDVI is compared to its own historical median for the same calendar month. The percentage deviation (anomaly %) is then classified using percentile-based thresholds computed once across the full 24-month analysis window:

    High Stress: anomaly % < P15
    Watch: P15 ≤ anomaly % < P33
    Normal: anomaly % ≥ P33

    Important limitations:
    • Vegetation condition classes indicate relative deviation from historical norms and do not represent absolute drought severity or crop failure.
    • This dashboard does not incorporate rainfall, soil moisture, or ground-truth crop data.
    • No yield inference, economic impact, or early warning alerts should be drawn from this data alone.
    • The historical baseline uses only 2 years of data; longer baselines improve reliability.
    • Ward boundaries shown are approximate sample polygons for development purposes.

    Technology

    100% open-source stack: Sentinel-2 (Copernicus), Python (geopandas, rasterio, rasterstats, numpy), HTML/CSS/JavaScript. No proprietary APIs or server-side code. All processing is reproducible from the accompanying Python scripts.
